URL shortening API landing page

This project was created for a challenge of front end mentor. The challenge is to integrate with the rel.ink API to create shortened URLs and display them like in the given design. I took this chance to get some more experience with asynchronous JavaScript.

Functional requirements: optimal layout depending on the device screen size, shorten any valid URL, see a list of their shortened links, even after refreshing the browser (local storage), copy the shortened link to their clipboard, receive error message when the input field is empty.

Technologies used:

HTML5 CSS3 CSS3: flexbox JavaScript API
